Explain why the alt attribute of the img tag is important.
In case the image doesn't come through, we have an alternative item to show. This also helps with programs that read webpages to blind individuals.

What is the purpose of name mangling?
It helps us distinguish between multiple items that may have the same name without the need to go out of our way to make them completely unique from one another.

Explain the difference between redirects that use a 301 HTTP status code and ones that use a 302 HTTP status code.
301 means that the page moved permanently, and that you shouldn't come back to this page, whereas 302 means the page has been moved temporarily, and to try this link again in the future.
